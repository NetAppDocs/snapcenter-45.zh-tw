= 在SnapCenter 還原伺服器中檢視匯入的備份
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
將SnapManager 您的支援範圍匯入Microsoft SQL Server歸檔備份之後、您可以SnapCenter 從*拓撲*頁面檢視這些備份。

*步驟*

. 在左導覽窗格中、按一下*資源*、然後從清單中選取適當的外掛程式。
. 建立包含所有資料庫的資源群組。
. 選取資源群組。
. 保護資源群組。
. 在「*拓撲*」頁面的「*管理複本*」檢視中、按一下「從Vault複本備份」。
+
此時會顯示備份的詳細資料。
