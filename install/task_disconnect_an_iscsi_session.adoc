= 中斷iSCSI工作階段的連線
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有時候、您可能需要中斷iSCSI工作階段與具有多個工作階段的目標之間的連線。

*步驟*

. 在左側導覽窗格中、按一下*主機*。
. 在「主機」頁面中、按一下「* iSCSI工作階段*」。
. 從*儲存虛擬機器*下拉式清單中、選取iSCSI目標的儲存虛擬機器（SVM）。
. 從*主機*下拉式清單中、選取工作階段的主機。
. 從iSCSI工作階段清單中、選取您要中斷連線的工作階段、然後按一下*中斷連線工作階段*。
. 在「中斷連線工作階段」對話方塊中、按一下「*確定*」。
